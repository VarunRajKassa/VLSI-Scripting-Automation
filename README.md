# Scripting Automation

## Project Overview

This repository contains a collection of Shell, Perl, TCL, and Python scripts developed for automation and problem-solving tasks relevant to VLSI design workflows. These scripts demonstrate practical applications such as file handling, pattern matching, regular expressions, data extraction, string manipulation, and process automation.

Scripting is an essential skill in semiconductor design flows for automating repetitive tasks, parsing reports, validating constraints, and integrating EDA tools efficiently.

---

## Languages Covered

### Shell Scripting
Used for Linux automation, file management, and process execution.

Implemented scripts:
- File existence checker
- Prime number checker
- Largest file analyzer

---

### Perl Scripting
Used for text processing, regular expressions, log parsing, and pattern matching.

Implemented scripts:
- Email validation
- Password strength validation
- Numeric data extraction

---

### TCL Scripting
TCL is one of the most important scripting languages in VLSI, widely used in EDA tools such as Cadence Innovus, Synopsys Design Compiler, PrimeTime, and Tempus.

Implemented scripts:
- Factorial calculator
- URL parser
- Associative array operations
- String reversal

---

### Python Scripting
Used for automation, utility scripting, and basic data manipulation.

Implemented scripts:
- Pattern generator
- List sorting
- Maximum and minimum finder
